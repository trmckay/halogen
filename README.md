# Halogen

[![Build status](https://drone.trmckay.com/api/badges/tm/halogen/status.svg)](https://drone.trmckay.com/tm/halogen)

A simple OS kernel for RISC-V made with Rust.

## Dev requirements

- `rustup`
- `riscv64-unknown-elf-gdb`
- `qemu-system-riscv64`
